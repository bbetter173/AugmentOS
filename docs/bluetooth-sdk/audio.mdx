The Bluetooth SDK exposes microphone state, raw audio callbacks, local transcription, and glasses media-volume behavior where supported by the connected model.

<Info>
Mentra Live and G2 both have microphones. Mentra Live has a speaker; G2 does not. Your app owns user-facing microphone permission copy, should ask for microphone access only when the feature needs it, and should provide a visible way to disable microphone streaming.
</Info>

## Enable Microphone Features

<Tabs>
  <Tab title="React Native">

```ts
await BluetoothSdk.setMicState({
  sendPcmData: true,
  sendTranscript: true,
  bypassVad: false,
  sendLc3Data: false,
});
```

  </Tab>
  <Tab title="Android">

```kotlin
sdk.setMicState(
    MicConfig(
        sendPcmData = true,
        sendTranscript = true,
        bypassVad = false,
        sendLc3Data = false,
    )
)
```

  </Tab>
  <Tab title="iOS">

```swift
sdk.setMicState(
    MicConfiguration(
        sendPcmData: true,
        sendTranscript: true,
        bypassVad: false,
        sendLc3Data: false
    )
)
```

  </Tab>
</Tabs>

## Local Transcription

<Tabs>
  <Tab title="React Native">

```ts
const sub = BluetoothSdk.addListener('local_transcription', (event) => {
  console.log(`${event.text} final=${event.isFinal}`);
});

sub.remove();
```

  </Tab>
  <Tab title="Android">

```kotlin
override fun onLocalTranscription(event: LocalTranscriptionEvent) {
    Log.d("Mentra", "${event.text} final=${event.isFinal}")
}
```

  </Tab>
  <Tab title="iOS">

```swift
func mentraBluetoothSDK(_ sdk: MentraBluetoothSDK, didReceive event: BluetoothEvent) {
    if case let .localTranscription(transcription) = event {
        print("\(transcription.text) final=\(transcription.isFinal)")
    }
}
```

  </Tab>
</Tabs>

## Raw Audio Callbacks

React Native can listen for microphone PCM events:

```ts
const sub = BluetoothSdk.addListener('mic_pcm', (event) => {
  console.log(event.pcm);
});
```

Native Android and iOS apps receive equivalent audio callbacks through the SDK listener/delegate surfaces. Disable raw callbacks when the app no longer needs them.

## Production Notes

- Always provide a user-visible microphone permission explanation.
- Let users disable microphone streaming.
- Expect model availability to differ by platform, locale, and app configuration.
- Keep cloud upload and retention policies explicit in your privacy disclosures.
- Disable raw audio callbacks when the app no longer needs them.
