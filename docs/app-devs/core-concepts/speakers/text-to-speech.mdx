---
title: "Text-to-Speech"
description: "Speak text aloud through the glasses speakers"
---

Use `session.speaker.speak()` to convert text to speech and play it through the glasses speakers. The audio plays on the user's phone, which routes it to the glasses.

## Basic Usage

```typescript
session.speaker.speak("Hello, welcome to my app");
```

The text is sent to the cloud, converted to speech, and played on the user's device.

## Common Patterns

### Respond to voice commands

```typescript
session.transcription.on(async (data) => {
  if (!data.isFinal) return;

  const text = data.text.toLowerCase();

  if (text.includes("weather")) {
    const weather = await fetchWeather();
    session.speaker.speak(`It's ${weather.temp} degrees and ${weather.condition}`);
    session.display.showReferenceCard("Weather", `${weather.temp}°F — ${weather.condition}`);
  }
});
```

### Confirm actions

```typescript
session.camera.takePhoto().then((photo) => {
  session.speaker.speak("Photo captured");
});
```

### Read notifications aloud

```typescript
session.phone.notifications.on((notification) => {
  session.display.showText(`${notification.app}: ${notification.title}`);
  session.speaker.speak(`New notification from ${notification.app}: ${notification.title}`);
});
```

## Tips

- Keep spoken text short and natural. The user is wearing glasses, not reading a document.
- Pair speech with a display update so the user gets both audio and visual confirmation.
- Don't call `speak()` on every interim transcription result — only on final results or specific triggers. Rapid-fire TTS calls will queue up and overlap.

## Migrating from v2

```typescript
// v2
session.audio.speak("Hello");

// v3
session.speaker.speak("Hello");
```

Same method signature, just on `session.speaker` instead of `session.audio`. See the [Migration Guide](/app-devs/migration/overview) for the full list of changes.
