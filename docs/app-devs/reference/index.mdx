# MentraOS SDK Reference

Welcome to the reference for the MentraOS SDK (`@mentra/sdk`). This section provides detailed information about all classes, interfaces, types, and enums.

## v3 SDK

The v3 SDK is organized around two main concepts:

- **[MiniAppServer](/app-devs/core-concepts/app-server)** — your app's entry point. Handles webhooks, WebSocket connections, and creates sessions.
- **[MentraSession](/app-devs/core-concepts/session)** — one user's connection to your app. All capabilities are accessed through 14 typed managers on the session object.

For a guide-oriented introduction, see [Core Concepts](/app-devs/core-concepts/app-server). For the full v2 → v3 method mapping, see the [API Map](/app-devs/migration/api-map).
